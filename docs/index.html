Static snapshot of medicaid-provider-spending.csv (238M rows, 2018–2024), pre-aggregated by month and HCPCS code. Provider breakdowns are available for the overall view and single-code views; the row table shows the 2,000 largest rows by payment. The raw data contains implausibly large payment values (single rows in the trillions) — use “Exclude extreme rows” to screen them out of the overall view. Blank NPIs appear as “—”. For full row-level and NPI-level exploration, run the local app in this repo.
